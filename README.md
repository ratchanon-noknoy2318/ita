# Legacy PHP System Debugging Experience

## Overview

During my internship at **Kosamphi Nakhon Hospital**, I worked on maintaining and debugging a **PHP legacy system** that was purchased for **9,000 THB**.

## Problem

- The system's main page returned a **404 error**.
- The issue was not caused by a single file but involved multiple parts of the application.

## Investigation & Solution

- Investigated the application's:
  - Routing configuration
  - Controllers
  - System configuration files
- Traced the source of the routing issue.
- Debugged and fixed **20+ files** across the system.

## Key Learnings

This experience improved my skills in:

- Legacy system maintenance
- Large-scale debugging
- Understanding existing codebases
- Real-world problem-solving
